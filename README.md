# Trayan Corporation Website

A modern website for Trayan Corporation, a chemical trading company, built with Next.js, Tailwind CSS, and TypeScript.

## Features

- Responsive design
- Modern UI with animations
- Product showcase
- Business information
- Contact details with map integration

## Prerequisites

- Node.js 18.x or later
- npm 9.x or later

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/yourusername/trayan-corp.git
cd trayan-corp
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `src/app/` - Next.js app router pages
- `src/components/` - Reusable components
- `public/` - Static assets

## Technologies Used

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Heroicons
